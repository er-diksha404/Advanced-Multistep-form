# 🚀 Advanced Form System

A multi-step form application built using React and Tailwind CSS.  
This project demonstrates real-world form handling with dynamic validation and reusable components.

---

## ✨ Features


- Multi-step form (Step 1 → Step 2 → Step 3)
- Form validation:
  - Full Name (min length)
  - Email (format validation)
  - Phone (10 digits only)
  - Age (18–110 range)
  - Address validation
- Dynamic form behavior:
  - Veg meal → shows Veg Type field
  - Infant selected → Baggage becomes optional
- Reusable components:
  - InputField
  - SelectField
- Custom Hook (`useForm`) for clean state management
- API simulation on form submission
- Form reset after submission
- Clean and responsive UI using Tailwind CSS

---

## 🛠 Tech Stack

- React (Vite)
- JavaScript (ES6+)
- Tailwind CSS
